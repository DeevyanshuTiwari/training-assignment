# 📦 Product Inventory Dashboard

## 📌 Project Overview
This is a **Product Inventory Dashboard** built using **HTML, CSS, and JavaScript**.

It allows users to:
- Add and delete products
- Search products instantly
- Filter by category and stock
- Sort products
- View analytics (total products, value, stock)
- Navigate using pagination

---

## 🚀 Features

- 🔍 Search products by name  
- 📂 Filter by category  
- ⚠️ Show low stock products  
- 🔃 Sort (price & name)  
- ➕ Add new product  
- ❌ Delete product  
- 📊 Inventory analytics  
- 📄 Pagination  
- 💾 Data stored using LocalStorage
